# Guy & Gal
Guy & Gal is a 2-player "game" developed in Monogame Extended.

Feel free to explore the source code!

------------------------------------------------------
# Controls

Player 1 - WASD and Left Shift (to sprint)

Player 2 - IJKL and Right Shift (to sprint)

ESC to exit the game
